## 🌟 What is ATS Resume?

> **ATS Resume** is a free, AI-powered web application built for students to create ATS-optimized resumes, check ATS scores, and land their dream jobs — completely free, no paywalls.

**The problem:** Tools like Resumeworded, Jobscan cost ₹2000/month. Students can't afford them.

**Our solution:** Build a better, completely free alternative with AI-powered features that don't exist anywhere else.

## ⚙️ Getting Started

### Prerequisites

```bash
Java 21+
Node.js 20+
MySQL 8.0+
Maven 3.8+
```

### Backend Setup

```bash
# Clone the repository
git clone https://github.com/sumitpandey7454/atsResume.git
cd atsResume/ats-resume-backend

# Create .env file
cp .env.example .env
# Fill in your credentials in .env

# Create MySQL database
mysql -u root -p
CREATE DATABASE atsresume;
exit;

# Run the application
./mvnw spring-boot:run
```

Backend runs on → `http://localhost:8080`

### Frontend Setup

```bash
cd ../ats-resume-frontend

# Install dependencies
npm install

# Create .env file
echo "VITE_API_URL=http://localhost:8080" > .env

# Run the application
npm run dev
```

Frontend runs on → `http://localhost:5173`

## 📡 API Endpoints

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| GET | `/api/auth/health` | Health check | ❌ |
| GET | `/api/auth/me` | Get user profile | ✅ |
| POST | `/api/ats/check` | Check ATS score (text) | ❌ |
| POST | `/api/ats/check-file` | Check ATS score (file) | ❌ |
| POST | `/api/resume/create` | Create resume | ✅ |
| GET | `/api/resume/all` | Get all resumes | ✅ |
| GET | `/api/resume/{id}` | Get resume by ID | ✅ |
| DELETE | `/api/resume/{id}` | Delete resume | ✅ |
| POST | `/api/resume/generate-about-me` | AI About Me | ✅ |
| POST | `/api/resume/generate-keywords` | AI Keywords | ✅ |
| POST | `/api/interview/generate` | Interview questions | ✅ |
| GET | `/api/templates/all` | Get all templates | ❌ |
| GET | `/api/admin/stats` | Admin stats | ✅ ADMIN |
